DLSYNTH_ERROR 

An unknown error has been encountered while processing the request.

No error has been encountered The file couldn't be opened The file was opened correctly, but contained invalid data The specified instrument doesn't exist The arguments that were passed were not valid The file uses an unsupported codec A condition has failed while loading the file
