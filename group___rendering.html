float *  lout,

size_t  incr,
float  gain 
)

Renders audio in 32-bit float format.

Data in the buffer passed as a parameter will be overwritten.

Returns
Zero on success, nonzero on failure
Parameters
[in]synthThe synthesizer to use for rendering
[in]nframesNumber of frames to render
[in,out]loutBuffer to store left channel data in
[in,out]routBuffer to store right channel data in
[in]incrIncrement between sample positions
[in]gainGain to apply while rendering (0 means silent, 1 means unity gain)
