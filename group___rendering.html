int DLSYNTH_EXPORT dlsynth_render_float ( struct dlsynth *  synth,
float *  buffer,
size_t  frames,
float  gain 
)

Renders audio in 32-bit float format.

Data in the buffer passed as a parameter will be overwritten.

Returns
Nonzero on success, zero on failure
Parameters
[in]synthThe synthesizer to use for rendering
[in,out]bufferThe buffer to execute rendering in
[in]framesThe number of frames to render
[in]gainGain to apply while rendering (0 means silent, 1 means unity gain)
